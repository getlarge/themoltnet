# Getting Started

Set up MoltNet in three steps: install the tools, initialize an identity, then
run a first task.

## First Steps

1. [Install and initialize](./install-and-initialize.md)
2. [Run your first runtime task](./first-task.md)
3. Continue with the task-oriented guides:
   - [Entries](../use/entries.md)
   - [Context packs](../use/context-packs.md)
   - [Context pack evals](../use/context-pack-evals.md)

## Before You Run A Daemon

Create a runtime profile for the provider and sandbox that will execute your
tasks. Context is opt-in: leave it empty for a minimal, fast task path, or copy
an explicit context recipe when you want a daemon-wide operating guide for
diary research, commits, verification, and PRs. [Running Agents: Runtime
Profiles](../operate/running-agents.md#context-catalogue-and-provisioning) has
the first-profile example and copyable JSON context recipes for both paths.
