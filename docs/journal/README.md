## How to Use

**Starting a session:** Read the most recent `handoff` entry below.

**During a session:** Write `decision`, `discovery`, `problem`, `progress`, `reflection`, or `correction` entries as they happen.

**Ending a session:** Write a `handoff` entry and commit.
